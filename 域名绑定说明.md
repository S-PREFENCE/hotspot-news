# PrenceYours 2026 - 免费固定域名配置指南

## 概述

通过 **ClouDNS 免费域名 + Cloudflare Tunnel** 实现永久固定的微信访问链接，完全免费。

**最终效果**：微信内打开 `https://你的域名.cloudns.cc` 即可访问，链接永远不变。

---

## 前提条件

1. 注册 ClouDNS 免费账号（https://www.cloudns.net/）
2. 注册 Cloudflare 免费账号（https://dash.cloudflare.com/sign-up）
3. 本机已下载 cloudflared.exe（放在项目目录下）

---

## 步骤一：在 ClouDNS 申请免费域名

1. 打开 https://www.cloudns.net/ ，点击注册
2. 注册并登录后，进入 **DNS Zones** → **Create Zone**
3. 选择 **Free Zone**（免费区域）
4. 输入你想要的域名前缀（如 `prenceyours`），选择免费后缀：
   - `.cloudns.cc`、`.cloudns.be`、`.cloudns.cx` 等
5. 点击创建，域名即时生效

**示例**：输入 `prenceyours`，选择 `.cloudns.cc`，最终域名为 `prenceyours.cloudns.cc`

---

## 步骤二：将域名托管到 Cloudflare

1. 登录 Cloudflare Dashboard（https://dash.cloudflare.com）
2. 点击 **添加站点**（Add a site）
3. 输入你在 ClouDNS 申请的域名（如 `prenceyours.cloudns.cc`）
4. 选择 **Free 免费计划**
5. Cloudflare 会给出两个 Nameserver 地址，类似：
   - `xxx.ns.cloudflare.com`
   - `yyy.ns.cloudflare.com`
6. **回到 ClouDNS**，进入你的域名管理页面
7. 找到 **Nameserver 设置**，将默认 Nameserver 改成 Cloudflare 给的那两个
8. 等待 DNS 生效（通常几分钟到几小时，最迟24小时）
9. Cloudflare Dashboard 上站点状态变为 **Active** 即生效

---

## 步骤三：登录 cloudflared 授权

1. 打开 CMD 或 PowerShell，进入项目目录：
   ```
   cd D:\A-ai自动化\hotspot_news
   ```

2. 运行登录命令：
   ```
   cloudflared.exe login
   ```

3. 浏览器自动打开，选择你刚才添加到 Cloudflare 的域名并授权

4. 授权成功后，凭证保存在 `C:\Users\ZhuanZ\.cloudflared\cert.pem`

---

## 步骤四：创建 Named Tunnel

1. 运行创建命令：
   ```
   cloudflared.exe tunnel create hotspot
   ```

2. 成功后输出 Tunnel ID（一串UUID），凭证文件保存在：
   `C:\Users\ZhuanZ\.cloudflared\hotspot.json`

3. 记下 Tunnel ID，后面要用

---

## 步骤五：配置 DNS 路由

运行以下命令将域名指向你的 tunnel：

```
cloudflared.exe tunnel route dns hotspot prenceyours.cloudns.cc
```

（把 `prenceyours.cloudns.cc` 替换为你实际申请的域名）

这会在 Cloudflare DNS 中自动添加一条 CNAME 记录。

也可以手动在 Cloudflare Dashboard → DNS 中添加：
- 类型：CNAME
- 名称：`@`（代表根域名）
- 目标：`<你的tunnel-id>.cfargotunnel.com`
- 代理状态：已代理（橙色云朵）

---

## 步骤六：更新配置文件

### 1. 更新 cloudflared-config.yml

把 `YOUR-TUNNEL-ID` 替换为步骤四中的 Tunnel ID，`YOUR-DOMAIN` 替换为你的域名：

```yaml
tunnel: 你的Tunnel-ID
credentials-file: C:\Users\ZhuanZ\.cloudflared\hotspot.json

ingress:
  - hostname: prenceyours.cloudns.cc
    service: http://localhost:5000
  - service: http_status:404
```

### 2. 更新 一键公网固定域名.bat

修改文件顶部的配置变量：
```batch
set TUNNEL_NAME=hotspot
set DOMAIN=prenceyours.cloudns.cc
```

## 常见问题

### Q: ClouDNS 免费 Zone 选哪个后缀？
A: 都可以，推荐 `.cloudns.cc`，最短最好记。

### Q: Nameserver 修改后多久生效？
A: 通常几分钟到几小时，最迟24小时。可以在 Cloudflare Dashboard 查看状态。

### Q: 访问显示 502 Bad Gateway
A: Flask 服务没启动，或端口不是 5000。检查 Flask 是否在运行。

### Q: 访问显示 1015 错误
A: Cloudflare 限流，等几分钟再试。

### Q: cloudflared 连接超时
A: 必须加 `--protocol http2` 参数！你的网络可能封了 QUIC 端口。

### Q: 微信分享不显示卡片
A: 微信需要获取 OG 标签，可能需要等几分钟缓存生效。

### Q: 临时链接方案（不想配置固定域名）
A: 双击 `一键公网访问.bat` 即可获得临时链接，但每次重启链接会变。
