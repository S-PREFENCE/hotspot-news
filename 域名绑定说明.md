# PrenceYours 2026 - 域名绑定操作说明

## 前提条件

1. 购买域名 `prenceyours2026.com`（推荐在 Cloudflare Registrar 或其他域名注册商购买，约 50-80元/年）
2. 注册 Cloudflare 免费账号（https://dash.cloudflare.com/sign-up）

---

## 步骤一：将域名托管到 Cloudflare

1. 登录 Cloudflare Dashboard（https://dash.cloudflare.com）
2. 点击 **添加站点**（Add a site）
3. 输入 `prenceyours2026.com`
4. 选择 **Free 免费计划**
5. Cloudflare 会给出两个 Nameserver 地址，类似：
   - `xxx.ns.cloudflare.com`
   - `yyy.ns.cloudflare.com`
6. 去你的域名注册商后台，把域名的 Nameserver 改成 Cloudflare 给的这两个
7. 等待 DNS 生效（通常几分钟到几小时）
8. Cloudflare Dashboard 上会显示站点状态变为 **Active**

---

## 步骤二：登录 cloudflared 授权

1. 打开终端（CMD 或 PowerShell），进入项目目录：
   ```
   cd D:\A-ai自动化\hotspot_news
   ```

2. 运行登录命令：
   ```
   cloudflared.exe login
   ```

3. 浏览器会自动打开，选择你的域名 `prenceyours2026.com` 并授权

4. 授权成功后，凭证文件会保存在 `C:\Users\ZhuanZ\.cloudflared\cert.pem`

---

## 步骤三：创建 Named Tunnel

1. 运行创建命令：
   ```
   cloudflared.exe tunnel create prenceyours2026
   ```

2. 成功后会输出 Tunnel ID（一串UUID），并生成凭证文件：
   `C:\Users\ZhuanZ\.cloudflared\prenceyours2026.json`

3. 记下 Tunnel ID，后面要用

---

## 步骤四：配置 DNS 路由

运行以下命令，将域名指向你的 tunnel：

```
cloudflared.exe tunnel route dns prenceyours2026 prenceyours2026.com
```

这会在 Cloudflare DNS 中自动添加一条 CNAME 记录：
- 名称：`prenceyours2026.com`
- 目标：`<tunnel-id>.cfargotunnel.com`

也可以手动在 Cloudflare Dashboard → DNS 中添加：
- 类型：CNAME
- 名称：`@`（代表根域名）
- 目标：`<tunnel-id>.cfargotunnel.com`
- 代理状态：已代理（橙色云朵）

---

## 步骤五：更新配置文件

确认 `cloudflared-config.yml` 中的 credentials-file 路径正确。

如果 tunnel ID 与配置中的不同，需要更新。可以用以下命令查看：
```
cloudflared.exe tunnel list
```

## 常见问题

### Q: 访问显示 502 Bad Gateway
A: Flask 服务没启动，或端口不是 5000。检查 Flask 是否在运行。

### Q: 访问显示 1015 错误
A: Cloudflare 限流，等几分钟再试。

### Q: 微信分享不显示卡片
A: 微信需要获取 OG 标签，可能需要等几分钟缓存生效。确保 index.html 中有正确的 og:title、og:description 标签。

### Q: 想换域名怎么办？
A: 修改 `cloudflared-config.yml` 中的 hostname，然后更新 DNS 记录。
