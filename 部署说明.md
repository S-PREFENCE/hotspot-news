## 🌍 部署到公网（微信可访问链接）

### 方式一：一键脚本（推荐）

1. 双击 `推送到GitHub.bat`，按提示操作
2. 脚本会引导你创建 GitHub 仓库并推送代码
3. 推送成功后，到 Render.com 部署（见下方）

### 方式二：手动操作

#### 第一步：在 GitHub 创建仓库

1. 打开 https://github.com/new
2. 仓库名填 `hotspot-news`
3. 选择 **Public（公开）**
4. **不要勾选** README / .gitignore / License
5. 点击 **Create repository**

#### 第二步：推送代码

在 PowerShell 中执行：

```powershell
cd "D:\A-ai自动化\hotspot_news"

"C:\Program Files\Git\bin\git.exe" remote remove origin 2>$null
"C:\Program Files\Git\bin\git.exe" remote add origin https://github.com/你的用户名/hotspot-news.git
"C:\Program Files\Git\bin\git.exe" branch -M main
"C:\Program Files\Git\bin\git.exe" push -u origin main
```

> 💡 如果要求输入密码，需要使用 Personal Access Token（不是 GitHub 登录密码）
> 生成地址：https://github.com/settings/tokens → Generate new token → 勾选 repo

#### 第三步：在 Render.com 部署

1. 访问 https://render.com 用 GitHub 账号登录（免费）
2. 点击 **New +** → **Web Service**
3. 选择 `hotspot-news` 仓库
4. 填写配置：
   - **Name**：hotspot-news
   - **Runtime**：Python 3
   - **Build Command**：`pip install -r requirements.txt`
   - **Start Command**：`gunicorn app:app`
   - **Instance Type**：选 **Free**
5. 点击 **Create Web Service**

等待 2~3 分钟，Render 会给你一个永久链接：
```
https://hotspot-news-xxxx.onrender.com
```

**✅ 这个链接发到微信里就能直接点开！**

## ❓ 常见问题

**Q：微信里打不开？**
A：确保使用的是 Render.com 的公网链接（https://...onrender.com），不是 localhost 地址。

**Q：首次打开很慢？**
A：Render 免费版 15 分钟无访问会休眠，首次冷启动约 30 秒。可以用 UptimeRobot 免费定时 ping 保持活跃。

**Q：怎么保持服务不休眠？**
A：注册 https://uptimerobot.com → 添加监控 → 输入你的 Render 链接 → 间隔 5 分钟。免费版支持 50 个监控。

**Q：更新代码后怎么重新部署？**
A：推送到 GitHub 后 Render 会自动重新部署（约 2 分钟）。
