## 🚀 本地使用（局域网内手机也能看）

### 方法一：双击启动（推荐）
直接双击 `启动服务.bat`，等待命令行出现：
```
Running on http://127.0.0.1:5000
Running on http://10.x.x.x:5000
```
然后：
- **电脑访问**：打开浏览器，输入 `http://localhost:5000`
- **同一 WiFi 的手机访问**：打开手机浏览器，输入命令行里显示的 `http://10.x.x.x:5000`（IP地址）

## 🌍 部署到云端（任何人都能访问）

需要先安装 Git，然后推送到 GitHub，再部署到 Render.com。

### 第一步：安装 Git

访问 https://git-scm.com/download/win 下载安装（一路默认即可）

安装完成后重新打开 PowerShell，验证：
```
git --version
```

### 第二步：初始化并推送代码到 GitHub

在 PowerShell 中执行以下命令：

```powershell
# 进入项目目录
cd "D:\A-ai自动化\hotspot_news"

# 初始化 Git
git init
git add .
git commit -m "初始化热点新闻网页项目"
```

然后去 https://github.com/new 创建一个新仓库，名称填 `hotspot-news`，**设为 Public（公开）**，**不要勾选** Add README。

创建完后，复制页面上的仓库地址，格式类似：
`https://github.com/你的用户名/hotspot-news.git`

然后继续执行：
```powershell
git remote add origin https://github.com/你的用户名/hotspot-news.git
git branch -M main
git push -u origin main
```

### 第三步：在 Render.com 部署

1. 访问 https://render.com 注册/登录（免费，可以用 GitHub 账号直接登录）
2. 点击 **New +** → **Web Service**
3. 选择 **Connect a repository**，授权 GitHub 后选择 `hotspot-news` 仓库
4. 填写配置：
   - **Name**：hotspot-news（随意）
   - **Runtime**：Python 3
   - **Build Command**：`pip install -r requirements.txt`
   - **Start Command**：`gunicorn app:app`
   - **Instance Type**：选 **Free**
5. 点击 **Create Web Service**

等待约 2~3 分钟部署完成，Render 会给你一个链接：
```
https://hotspot-news-xxxx.onrender.com
```

**这个链接就是你的公开网址，任何人、任何设备都能访问！**

## ❓ 常见问题

**Q：命令行关了，网页就打不开了？**
A：是的，本地运行时命令行不能关。如果想关掉命令行，使用"后台启动并打开浏览器.bat"，或者部署到 Render.com。

**Q：知乎数据有时抓不到？**
A：知乎反爬较严，微博+百度两个平台已能保证20条热点，知乎作为补充来源。

**Q：Render.com 免费版有什么限制？**
A：免费版15分钟无访问会休眠（首次访问会慢30秒），有访问时正常运行。可以用免费的 UptimeRobot 每10分钟 ping 一次，保持服务不休眠。
